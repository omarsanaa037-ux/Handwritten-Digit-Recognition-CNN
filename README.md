# Handwritten Digit Recognition using CNN

A professional Deep Learning project developed in Google Colab that utilizes **Convolutional Neural Networks (CNN)** built with **TensorFlow** and **Keras** to recognize handwritten digits from the famous MNIST dataset.

## 🚀 Features
- High-accuracy CNN architecture (Conv2D, MaxPooling2D, Dropout, Dense).
- Real-time prediction visualization using `matplotlib`.
- Custom image processing using `OpenCV` (Resizing, Inverting, Gaussian Blur, Otsu's Thresholding) to allow users to upload and test their own handwritten drawings.
- EarlyStopping implemented to prevent overfitting and optimize training time.

## 🛠️ Tech Stack
- **Language:** Python 3
- **Frameworks:** TensorFlow, Keras
- **Libraries:** NumPy, OpenCV (cv2), Matplotlib
- **Environment:** Google Colab / Jupyter Notebook
